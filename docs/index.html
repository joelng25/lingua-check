Política de privacidad de LinguaCheck

Última actualización: 13 de julio de 2026

LinguaCheck es una extensión de Chrome que revisa ortografía y gramática mientras escribes en sitios web. Esta política explica qué datos se procesan y cómo.

1. Responsable

El responsable del tratamiento es el desarrollador de LinguaCheck. Para consultas de privacidad: joelnogao625@gmail.com

2. Qué datos se procesan

  • Texto que escribes en campos editables (inputs, áreas de texto, editores compatibles) cuando la extensión está activa.
  • Preferencias guardadas en tu navegador: idioma, diccionario personal, reglas ignoradas y sitios desactivados.

3. Cómo se usan los datos

  • El texto se envía a la API de LanguageTool (o a un servidor personalizado que configures) únicamente para obtener sugerencias de corrección.
  • Las preferencias se almacenan localmente con chrome.storage.sync de Google Chrome. No se venden ni se comparten con terceros con fines publicitarios.

4. Servidores de terceros

Por defecto, LinguaCheck usa la API pública de LanguageTool (api.languagetool.org). Si activas un servidor propio en las opciones, el texto se enviará a esa URL. Consulta la política de privacidad de LanguageTool en languagetool.org/legal/privacy.

5. Qué no recopilamos

  • No creamos cuentas de usuario.
  • No recopilamos historial de navegación completo.
  • No vendemos datos personales.
  • No usamos el texto para entrenar modelos propios.
  • No recopilamos nombre, dirección ni datos de pago.

6. Permisos de la extensión

  • storage — guardar configuración y diccionario.
  • tabs — mostrar el número de errores por pestaña.
  • sidePanel — panel lateral con la lista de errores.
  • host_permissions — conectar con el servicio de corrección.
  • content_scripts — revisar solo los campos de texto donde escribes.

7. Conservación

El texto no se almacena de forma permanente en nuestros servidores (no operamos backend propio). Las preferencias permanecen en tu navegador hasta que las borres o desinstales la extensión.

8. Tus derechos

Puedes desactivar LinguaCheck en cualquier momento, excluir sitios concretos, borrar tu diccionario personal desde las opciones o desinstalar la extensión.

9. Menores

LinguaCheck no está dirigida a menores de 13 años y no recopilamos datos de menores de forma consciente.

10. Cambios

Podemos actualizar esta política. La fecha de la última revisión aparecerá al inicio del documento.
